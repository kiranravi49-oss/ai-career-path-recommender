# 🚀 AI Career Path Recommender

## 📌 Overview
The AI Career Path Recommender helps mid-career professionals identify their next career move based on their current role, experience, and skills.

It provides intelligent recommendations, skill gap analysis, and a clear career roadmap.

---

## 🔥 Key Features

- ✅ Career path recommendations (Top 3)
- ✅ Skill gap analysis
- ✅ Career roadmap (step-by-step learning)
- ✅ AI-based insights
- ✅ Simple and interactive UI

---

## 🧠 How It Works

1. User inputs:
   - Current role
   - Experience level
   - Skills

2. System:
   - Matches skills with predefined career paths
   - Calculates match score
   - Suggests best next roles

3. Output:
   - Top 3 career options
   - Recommended skills
   - Career roadmap
